# Components compiler

Webpack compiler for creating frontend component. Build on top Twig templates, Scss and ES6 javascript.

## How to start

```
npm install
```

```
npm run server 
```
starts server on port 8001

```
npm run build
```
build standalone HTML/CSS/JS component

## Webpack loaders

- twig-loader
- babel-loader with ES2015 preset
- scss-loader
- css-loader
